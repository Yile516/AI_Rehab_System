# 如何使用自己的影片訓練 AI 模型

您可以透過錄製真實的測試影片，讓 AI 學習判斷標準。請依照以下步驟操作：

## 1. 準備影片
請準備三個類別的影片，分別放入對應的資料夾中：

*   **`training_data/0_healthy` (健康)**
    *   放入動作標準、健康的影片。
    *   特徵：起立速度快、軀幹無過度前傾、手抱胸好好的。
*   **`training_data/1_frailty` (虛弱)**
    *   放入模擬老人家虛弱的影片。
    *   特徵：起立很慢、為了站起來身體往前甩很大力。
*   **`training_data/2_compensation` (代償)**
    *   放入手扶膝蓋的影片。
    *   特徵：起立時手去撐膝蓋或大腿。

> **注意**：影片最好只包含單次起立坐下動作，且全身都要拍進去 (側面視角最佳)。

## 2. 執行訓練
在終端機 (Terminal) 執行以下指令：

```bash
python model/train_from_videos.py
```

## 3. 等待完成
程式會自動分析所有影片，提取特徵並重新訓練 `model/rehab_model.pkl`。
完成後，重啟網頁伺服器 (`python app.py`)，新的判斷標準就會生效！
